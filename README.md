# 🏨 Hotel Booking Cancellation Prediction using Machine Learning

## 📌 Project Overview
This project analyzes hotel booking data and predicts whether a booking will be canceled using Machine Learning techniques.

---

## 🎯 Objectives
- Understand dataset and features  
- Perform data cleaning  
- Handle missing values  
- Perform EDA  
- Build ML models  
- Evaluate performance  
- Predict cancellations  

---

## 📊 Dataset Information
- Records: ~119,000  
- Features: 30+  
- Target: `is_canceled`  

---

## 🧹 Data Preprocessing
- Removed unnecessary columns  
- Handled missing values  
- Encoded categorical data  
- Train-test split  
- Feature scaling
